# Deterministic Semantic Forge

Use this skill for rule-based concept mining and study artifact generation.

## Focus

- salience ranking from source text
- instructor-emphasis weighting
- confusion pair mining
- relationship mapping
- drill and trap generation
- provenance logging for every derived artifact

## Workflow

1. Start from normalized text and explicit thresholds.
2. Prefer transparent scoring formulas over magic heuristics.
3. Keep derived outputs explainable from the source.
4. Add fixtures when thresholds or ranking behavior change.
