# UI Systems Polish

Use this skill when refining ATLAS or extension UI surfaces.

## Focus

- typography scale and spacing rhythm
- card and panel consistency
- focus visibility
- reduced motion and contrast support
- keyboard navigation
- desktop/mobile layout priorities

## Rule

The UI should feel premium because the information architecture is strong, not because motion is compensating for weak structure.
