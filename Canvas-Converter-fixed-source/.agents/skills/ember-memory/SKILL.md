# Ember Memory

Use this skill when implementing or extending local memory that improves later deterministic runs.

## Focus

- concept aliases approved by the user
- confusion patterns
- selector confidence history
- instructor emphasis phrases
- user correction memory

## Rule

EMBER is deterministic memory, not model retraining. Every stored signal should have a clear write path, read path, and scope boundary.
